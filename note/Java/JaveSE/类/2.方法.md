## 重载

1. **方法名相同**
2. 方法的**参数类型，参数个数**不一样
3. 方法的返回类型可以不一样
4. 方法的修饰符可以不相同

## 可变参数

1. 在指定类型后加一个省略号(...)
2. 一个方法中只能指定一个可变参数，它【必须是方法的最后一个参数】
3. 可变参数列表是数组
